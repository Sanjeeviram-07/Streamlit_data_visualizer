#  Smart Data Visualizer

An interactive data visualization dashboard built using Streamlit that
allows users to upload CSV datasets and generate dynamic visual insights
instantly.

🔗 **Live App:** 

------------------------------------------------------------------------

##  Features

-   Upload CSV files
-   Dataset preview
-   Summary statistics
-   Line Chart
-   Bar Chart
-   Scatter Plot
-   Histogram
-   Correlation Heatmap
-   Download processed data

------------------------------------------------------------------------

##  Tech Stack

-   Python 3.11
-   Streamlit
-   Pandas
-   Plotly
-   Matplotlib
-   Seaborn

------------------------------------------------------------------------

##  Run Locally

``` bash
pip install -r requirements.txt
streamlit run app.py
```

##  Use Case

This application helps users quickly analyze and visualize structured
datasets without writing code, making it useful for:

-   Students
-   Data Analysts
-   Beginners in Data Science
-   Quick exploratory data analysis tasks
